# ☕ Coffee House

A premium, responsive, and visually stunning landing page for a modern specialty coffee shop. Built with high-end aesthetics, smooth animations, and a seamless user experience.

**[🌐 Live Demo](https://prabhatkumargupta00.github.io/coffee-house/)**

## ✨ Features

- **Premium Design**: Modern dark theme with glassmorphism effects and elegant typography.
- **Interactive Menu**: Explore our signature blends with dynamic hover effects and "Add to Cart" functionality.
- **Responsive Layout**: Optimized for all devices—from desktop monitors to mobile screens.
- **Smooth Navigation**: One-page architecture with smooth scrolling to sections like Home, Menu, About, and Reviews.
- **Table Booking**: Seamless "Book Table" integration with a streamlined form and toast notifications.
- **Swiper Integration**: Beautiful hero carousel featuring high-quality coffee imagery.

## 🛠️ Built With

- **HTML5**: Structural markup and SEO.
- **Vanilla CSS**: Premium styling.
- **JavaScript**: Interactive logic and cart management.
- **Swiper.js**: Professional slider for the hero section.
- **Font Awesome**: Premium looking icons.
- **Google Fonts**: Modern looking fonts.

## 🚀 Getting Started

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/prabhatkumargupta00/coffee-house.git
    ```
2. Navigate to the project directory:
    ```bash
    cd coffee-house/Coffee
    ```
3. Open `index.html` in your browser.
